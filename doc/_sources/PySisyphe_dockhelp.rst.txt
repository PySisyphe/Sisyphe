.. _page-dockhelp:

Documentation tab of the dock
=============================

An HTML client for viewing PySisyphe documentation.

A button bar at the top:

|home|
	Return to start page.

|left| 
	Go to previous page.

|right| 
	Go to next page.
	
**Search** edit box: edit keyword(s) to be searched in the documentation. The results are displayed as links in the HTML client.
